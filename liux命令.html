xargs ​

将标准输入作为命令行入参

参数： ​

  • -p 参数打印出要执行的命令，询问用户是否要执行
  • -t 参数则是打印出最终要执行的命令，然后直接执行，不需要用户确认。

例子 ​

git branch | grep "xxx" | xargs git checkout
